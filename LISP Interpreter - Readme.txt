How to run this project submission.?

Here is the solution:

1) Run make

   This should build all the .java files and generate necessary .class files.

2) java MyInterpreter < input_test_file_name
 
   Please run this command. 
   sample run command: java MyInterpreter < sampleinput.txt (sampleinput.txt file contains a sample input S-expression in it)

cleaning all class files:--

1) Run make clean

  This would delete all the .class files.
